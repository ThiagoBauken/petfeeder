Pinagem dos Motores (ULN2003)

Cada compartimento usa 1 motor. Comece com 1 motor e adicione mais conforme necessidade.

Motor 1 - Compartimento 1 Obrigatório

ULN2003 ESP32 Descrição
IN1 GPIO 16 Bobina A
IN2 GPIO 17 Bobina B
IN3 GPIO 18 Bobina C
IN4 GPIO 19 Bobina D
VCC 5V Alimentação motor
GND GND Terra comum

Motor 2 - Compartimento 2 Opcional

ULN2003 ESP32 Descrição
IN1 GPIO 25 Bobina A
IN2 GPIO 26 Bobina B
IN3 GPIO 27 Bobina C
IN4 GPIO 32 Bobina D
VCC 5V Alimentação motor
GND GND Terra comum

Motor 3 - Compartimento 3 Opcional

ULN2003 ESP32 Descrição
IN1 GPIO 12 Bobina A
IN2 GPIO 13 Bobina B
IN3 GPIO 14 Bobina C
IN4 GPIO 15 Bobina D
VCC 5V Alimentação motor
GND GND Terra comum
Atenção: GPIO 12 pode causar problemas no boot se estiver HIGH. Certifique-se de que o motor está em LOW ao iniciar.

Sensor Ultrassônico HC-SR04

HC-SR04 ESP32 Descrição
VCC 5V Alimentação (5V obrigatório!)
GND GND Terra comum
TRIG GPIO 26 Disparo do pulso
ECHO GPIO 25 Recepção do eco

Como funciona a detecção de nível?

O sensor fica no topo do reservatório de ração, apontando para baixo. Ele mede a distância até a superfície da ração:

Dica: Você pode ajustar os valores de 5cm e 30cm no código conforme o tamanho do seu reservatório.

Configuração de Doses

O motor de passo 28BYJ-48 tem 2048 passos por volta completa. As doses são configuradas assim:

Dose Passos Gramas (aprox) Voltas
Pequena 2048 ~50g 1 volta
Média 4096 ~100g 2 voltas
Grande 6144 ~150g 3 voltas
Calibração: A quantidade de gramas depende do mecanismo do seu alimentador. Faça testes e ajuste os valores no código.

Código para ajustar doses:

// No arquivo PetFeeder_SaaS_MultiMotor.ino const int DOSE_SMALL = 2048; // 1 volta (~50g) const int DOSE_MEDIUM = 4096; // 2 voltas (~100g) const int DOSE_LARGE = 6144; // 3 voltas (~150g) // Para adicionar mais motores, altere: #define NUM_COMPARTMENTS 1 // Mude para 2 ou 3

Passo a Passo

Monte as conexões

Siga o diagrama acima para conectar o sensor HC-SR04 e o(s) motor(es) ao ESP32.

Carregue o código no ESP32

Abra o Arduino IDE, selecione "ESP32 Dev Module" e faça upload do arquivo PetFeeder_SaaS_MultiMotor.ino

Conecte na rede WiFi do ESP32

O ESP32 criará uma rede chamada "PetFeeder_XXXXXX". Conecte-se com a senha 12345678

Acesse o portal de configuração

Abra o navegador e vá para http://192.168.4.1

Configure o WiFi e token

Selecione sua rede WiFi, digite a senha e cole o token obtido na página "Minha Conta" do sistema.

Pronto!

O ESP32 irá reiniciar e conectar automaticamente. O dispositivo aparecerá no seu dashboard.

Sucesso! Seu PetFeeder está configurado e pronto para uso. Adicione seus pets e crie horários de alimentação!

Solução de Problemas

Motor não gira:
  • Verifique todas as 4 conexões IN1-IN4
  • Confirme que VCC do ULN2003 está em 5V
  • Verifique se o GND está conectado
  • Teste trocar a ordem dos pinos (IN1↔IN3, IN2↔IN4)
Sensor não funciona:
  • O VCC DEVE ser 5V (não funciona em 3.3V)
  • Troque TRIG e ECHO se as leituras forem zero
  • Evite superfícies muito próximas (<2cm) ou inclinadas
ESP32 não conecta ao WiFi:
  • Verifique se a senha está correta
  • O ESP32 só suporta WiFi 2.4GHz (não 5GHz)
  • Tente aproximar do roteador
