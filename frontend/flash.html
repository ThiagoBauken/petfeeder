Instalar Firmware PetFeeder

Conecte seu ESP32 via cabo USB e clique no botao abaixo.
O firmware sera baixado e gravado automaticamente.

Como funciona:

1

Conecte o ESP32

Use um cabo USB de dados (nao apenas de carga)

2

Clique em "Instalar Firmware"

O navegador vai pedir para selecionar a porta serial

3

Selecione a porta COM

Geralmente aparece como "CP2102" ou "CH340"

4

Aguarde o flash (~60 segundos)

O ESP32 sera reiniciado automaticamente

Requisitos:

  • Google Chrome, Edge ou Opera (desktop)
  • Driver USB instalado (CP2102 ou CH340)
  • Cabo USB de dados
  • ESP32 DevKit
